ねこのようちえん レスポンシブ版（日本語・旧サイト参考版）

この版では、旧サイト https://nekohoiku.com/ の主な構成を参考にして、
スマホで見やすく、更新しやすい形へ整理しました。

入っている主なページ
- index.html           トップ
- profile.html         プロフィール
- guide.html           ご利用案内
- cats.html            里親募集中
- contact.html         ご予約 / お問い合わせ
- faq.html             よくある質問
- admin.html           里親情報更新ページ

ポイント
1. 旧サイトの雰囲気を残しつつ、スマホ表示を改善
2. 里親募集ページは admin.html から更新可能
3. HTMLを直接触らなくても、一覧更新の練習がしやすい

注意
- admin.html はブラウザ保存型です
- 別PCに自動同期はされません
- 正式運用する場合は、Google Sheet連携などに進めるとさらに便利です

- メールアドレス
- お問い合わせフォームURL
- 実際の里親募集状況
- 必要に応じたFAQ文言
